# Characters

Cross-media canon for characters shared across Tepenian Universe projects (InnerTepeniaGDD, TheCryptographHelixDD, OuterTepenia1_GDD, and future projects).

Copy `z-template/` to start a new character. Keep entries here scoped to facts that hold true regardless of medium — origin, timeline placement, relationships, loyalties. Project-specific mechanics, stats, and story-craft frameworks belong in that project's own repo.
